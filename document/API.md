| ------------ | ------------ | ------------ | --------------------------------------- |
| Id           | Id           | string       |                                         |
| Name         | イベント名称 | string       |                                         |
| Date         | 開催日時     | date         |                                         |
| ClosingDate  | 出欠締切日時 | date         |                                         |
| Prticipants  | 参加者一覧   | コレクション |                                         |

| ...CreatedAt | 登録日       | date         |                                         |
| ...Status    | 出欠         | string       | 出席(attend),欠席(absence),未登録(none) |
